# Bamo - Descubre rutas de bus en tiempo real

Bamo es una aplicación web moderna diseñada para ayudar a los usuarios en Barranquilla/Buenos Aires a encontrar rutas de bus en tiempo real de manera eficiente y sencilla.

## Tecnologías Utilizadas

- **Frontend**: React + TypeScript + Vite
- **Estilos**: Vanilla CSS (Variables globales y diseño responsivo)
- **Control de Versiones**: Git & GitHub

## Requisitos Previos

Para ejecutar este proyecto en tu máquina local, necesitas tener instalado:

1.  **Node.js**
2.  **Git**

## Instalación y Uso

Sigue estos pasos para configurar el proyecto en una nueva computadora:

1.  **Clonar el repositorio**:
    ```bash
    git clone https://github.com/Emmanuel1103/BamoCo.git
    ```
2.  **Navegar al directorio del frontend**:
    ```bash
    cd BamoCo/frontend
    ```
3.  **Instalar dependencias**:
    ```bash
    npm install
    ```
4.  **Iniciar el servidor de desarrollo**:
    ```bash
    npm run dev
    ```

## Características principales

- **Hero Section**: Diseño de dos columnas con propuesta de valor clara y call-to-action.
- **Header Responsivo**: Menú de navegación con botón hamburguesa animado para dispositivos móviles.
- **Scroll Suave**: Navegación fluida entre las diferentes secciones de la landing page.
